# Ancadeba

## Project Overview
Ancadeba is a modular TypeScript game engine focused on clarity and rapid iteration. It provides the core building blocks for experimenting with gameplay ideas and includes an editor for content creation.

## Goals
- Offer a lightweight game engine built with modern web tooling.
- Encourage experimentation and learning through a simple architecture.
- Deliver a foundation that can be extended for custom projects.

## Prerequisites
- **Node.js** 18 or higher
- **npm** (comes with Node.js)
- Recommended: a code editor such as VS Code and Git for version control.

## Setup
Clone the repository and install dependencies:

```bash
npm install
```

Start the development environment:

```bash
npm run dev
```

This will launch the engine in development mode and watch for changes.

## Testing and Build Commands
Run the following commands before submitting changes:

```bash
npm run lint     # static analysis
npm test         # unit tests
npm run build    # production build
```

## Contribution Guidelines
1. Fork the repository and create feature branches for your work.
2. Ensure code adheres to the project's linting rules and passes all tests.
3. Submit a pull request with a clear description of your changes.
